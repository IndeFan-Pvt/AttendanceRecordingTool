共有用

共有用フォルダの使い方

このフォルダをそのまま相手に渡してください。配布物本体、勤怠シート、説明書を 1 つにまとめています。

含まれているもの

  • generate_akanecco_shift_gui: GUI 配布物本体です。
  • 勤怠データ: 勤怠表ファイルをまとめたフォルダです。現在は 【統一書式】あかねっこ_設定項目追加.xls を入れています。
  • あかねっこ_設定項目追加_説明書.html: Excel 側の設定欄の見方を説明した資料です。

相手に案内する内容

  1. generate_akanecco_shift_gui/generate_akanecco_shift_gui.exe を起動します。
  2. 記載したい勤怠表として 勤怠データ/【統一書式】あかねっこ_設定項目追加.xls を選びます。
  3. 設定 JSON は generate_akanecco_shift_gui/shift_config.json を選びます。
  4. 必要なら前月勤務表も選びます。
  5. Excel 側の設定欄を確認したい場合は あかねっこ_設定項目追加_説明書.html を開きます。

同梱している勤怠データ

  • 記載したい勤怠表: 【統一書式】あかねっこ_設定項目追加.xls
  • 必要なら同月の参照元勤務表を追加してください。例: 【統一書式】あかねっこ2月.xls
  • 必要なら前月勤務表を追加してください。例: 【統一書式】あかねっこ1月.xls

勤怠データ フォルダには、記載したい勤怠表と、必要に応じて同月参照元勤務表・前月勤務表を入れてください。

GUI の詳細手順は 3分で使う GUI版 と 別PC移植手順 を参照してください。
